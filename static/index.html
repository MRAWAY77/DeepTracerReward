What this dataset is

DeepTraceReward is a fine-grained benchmark and training set for human-perceived fakeness in generated video. A trace couples a natural-language reason with a spatial bounding box and a temporal interval.

The local release has 3,318 generated videos, 3,318 matched real clips, and 4,334 trace annotations. Multiple trace rows can point to one fake video.

Important caveats

The nine paper categories are normalized here from the release’s noisier, multi-label Labelbox tags. Use them for exploration, not as a lossless official relabeling. Blank explanations and tags are intentional missing annotations.
